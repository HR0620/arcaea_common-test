# Arcaea共通テスト 公式サイト

Arcaeaの非公式共通テストの公式サイト（試験運用版）です。
大学入試センター試験や共通テストの公式サイトを模した、シンプルかつ信頼感のあるデザインを採用しています。

## 機能概要

### 1. トップページ (`index.html`)
- **実施中の試験**: 現在実施されている試験の情報を表示。
- **新着情報**: お知らせリスト。
- **過去の試験問題**: PDFファイルへのリンクを提供。
- **試験結果 (Results)**: テーブル形式で平均点や標準偏差を表示。
- **Q&A**: よくある質問をアコーディオン形式で表示。

### 2. スタイル (`css/style.css`)
- 変数 (`:root`) テーマカラー等を管理。
- レスポンシブ対応 (スマホ・PC)。

### 3. スクリプト (`js/script.js`)
- ハンバーガーメニューの開閉。
- Q&Aの開閉アニメーション。

## メンテナンス方法

### 新しい試験を追加する場合
1. PDFファイルを `assets/pdfs/` にアップロードします (例: `exam2.pdf`)。
2. `index.html` の `exam-grid` 内に新しい `.card` 要素を追加し、リンクを設定します。

### ニュースを更新する場合
`index.html` の `.news-list` 内に新しい `<li>` を追加してください。
